# 好兄弟牌压缩工具 - 给最爱的电影瘦瘦身

基于 FFmpeg 的批量视频压缩工具，**内置 FFmpeg，无需额外安装**，支持硬件加速，保持原文件夹结构。

## 功能特性

- **内置 FFmpeg**：EXE 已包含 FFmpeg，可在任何 Windows 电脑上运行
- **批量压缩**：递归扫描整个文件夹及子文件夹
- **硬件加速**：自动检测 NVIDIA/AMD/Intel 硬件编码器，逐文件尝试硬件编码，失败自动回退软件编码
- **动态参数切换**：根据编码器类型自动切换质量参数（软件 CRF / NVIDIA CQ / Intel GQ / AMD QP），并提供默认值
- **分辨率调整**：支持原分辨率、720p、1080p、2K、4K
- **音频控制**：支持直接复制原音频流或指定 AAC 比特率
- **编码格式**：H.265 (HEVC) 或 H.264 (AVC)
- **压缩控制**：支持暂停/继续、停止（真正挂起 FFmpeg 进程）
- **实时进度**：进度条显示当前视频处理进度、帧数、速度、剩余时间
- **中文支持**：完全支持中文路径、特殊字符文件名、超长路径
- **压缩统计**：完成后显示原始总大小、压缩后总大小、节省空间及百分比
- **保持结构**：严格保持原文件目录结构

```

|  +---------------+          +---------------+             |
|  |  原视频文件夹 |          | 压缩后文件夹 |                |

|  | 子目录A       |<-------->| 子目录A       |              |
|  |   ├ video1.mp4|          |   ├ video1.mp4|             |
|  |   └ video2.mp4|          |   └ video2.mp4|             |
|  +---------------+          +---------------+             |
|  | 子目录B       |<-------->| 子目录B       |              |
```

## 使用说明

### 1. 选择原视频文件夹
点击「浏览」选择包含视频的文件夹，工具自动递归扫描所有子文件夹。

### 2. 选择压缩输出文件夹
选择压缩后视频保存位置，工具自动创建与原文件夹相同的目录结构。

### 3. 调整压缩参数
- **质量参数**：根据编码器自动切换，可直接使用默认值
  - 软件编码：CRF 28-35（默认 28）
  - NVIDIA 硬件：CQ 30-40（默认 30）
  - Intel 硬件：GQ 30-45（默认 30）
  - AMD 硬件：QP 24-34（默认 24）
- **分辨率**：原分辨率、720p、1080p、2K、4K
- **音频比特率**：不压缩（直接复制）或指定码率
- **硬件加速**：默认开启，自动选择最优编码器
- **编码格式**：H.265 (HEVC) 或 H.264 (AVC)

### 4. 开始压缩
点击「开始压缩」，信息区域会实时显示压缩进度和编码器参数。

## 代码结构

所有可配置参数位于代码头部，包括：
- 窗口标题、尺寸、背景色
- 所有界面元素的颜色、字体、尺寸、位置
- 默认压缩参数（CRF/CQ/GQ/QP、编码、分辨率、音频）
- 下拉菜单选项
- 支持的视频格式列表

主要类：
- `FFmpegManager` - FFmpeg 路径管理（内置释放 + 外部检测）
- `HardwareDetector` - 硬件加速检测
- `FileHandler` - 文件处理（扫描、路径管理、时长探测）
- `FFmpegCommandBuilder` - FFmpeg 命令生成
- `CompressThread` - 压缩线程（后台执行，不阻塞 UI）
- `VideoConverterGUI` - 主界面类

## 注意事项

1. 首次启动 EXE 需释放内置 FFmpeg 到临时目录，约需 5-10 秒
2. 硬件加速逐文件尝试，单个文件失败自动回退软件编码，不影响其他文件
3. 已存在的输出文件会被跳过
4. 压缩失败的文件会记录在日志中，并显示具体错误原因
5. 支持中文路径、特殊字符文件名和超长路径
6. 暂停功能真正挂起 FFmpeg 进程，继续后从暂停位置恢复
